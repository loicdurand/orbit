@specify project_definition # IDENTITÉ DU PROJET Nom: Orbit Type: Application Mobile (iOS/Android) Concept: Un "CRM Personnel" et assistant de mémoire sociale respectueux de la vie privée. Mission: Aider l'utilisateur à entretenir des relations de qualité avec ses 150 proches (Nombre de Dunbar) en réduisant la charge mentale. # CIBLES UTILISATEURS 1. Le "Networker Débordé" : A besoin de se souvenir de détails précis pour maintenir son réseau pro/perso. 2. L'Ami Distrait (Moi) : Oublie les dates et les détails de vie, veut un "second cerveau" pour être plus attentionné. # FONCTIONNALITÉS CLES (MVP - Minimum Viable Product) 1. "Quick Capture" (Capture Rapide) : - Une interface minimaliste pour saisir une note textuelle ou vocale après une rencontre. - Ex: "J'ai vu Marc, sa fille Léa rentre au CP demain, il aime le Whisky japonais." 2. Traitement IA (Le Cœur du système) : - L'application analyse la note brute pour extraire et structurer les données : - Mises à jour de la fiche contact (Enfant: Léa, Intérêt: Whisky). - Création de rappels contextuels (Rappel demain: Demander comment s'est pasée la rentrée de Léa). - Suggestion de "Ice-breakers". 3. Tableau de Bord "Jardinier" : - Vue non pas par liste alphabétique, mais par "besoin d'arrosage" (contacts négligés depuis trop longtemps). - Indicateurs visuels de la santé de la relation. # STACK TECHNIQUE (IMPERATIF) - Framework : Flutter (Dart) pour une performance native cross-platform. - Architecture : LOCAL-FIRST. - Base de données : SQLite (via Drift) ou Isar. Les données personnelles restent sur le téléphone. - IA : Intégration API (OpenAI ou Claude) pour le traitement des textes, mais anonymisation des données avant envoi si possible, ou traitement local si le device le permet. # DIRECTIVES DE DESIGN & UX - Vibe : Calme, sereine, pas de notifications agressives. - Style : Néomorphisme léger ou Minimaliste "Clean". - Priorité : Rapidité de saisie (moins de 3 secondes pour ouvrir et noter). # OBJECTIF DU REVENU - Structure du code prévue pour un modèle Freemium (Paywall pour +15 contacts et fonctionnalités IA avancées).
